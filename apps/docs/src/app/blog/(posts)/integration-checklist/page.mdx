# Integration Checklist for Production

Before going live, confirm each item below.

## Environment and credentials

- Store merchant credentials in server-only environment variables.
- Keep callback URLs consistent between dashboard and deployment.

## Transaction flow

- Validate every callback payload.
- Reconcile status using backend verification before marking payment complete.

## Monitoring

- Log transaction IDs and statuses.
- Add alerting for callback failures and signature mismatch events.
