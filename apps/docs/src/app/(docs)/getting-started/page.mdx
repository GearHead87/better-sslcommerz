# Getting Started

This page shows the quickest path to set up Better SSLCommerz.

## Installation

```bash
pnpm add @better-sslcommerz/better-sslcommerz
```

## Basic setup

Create your payment client, then call it from a server route.

```ts
import "@better-sslcommerz/better-sslcommerz";
```

## Next steps
